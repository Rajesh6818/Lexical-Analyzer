# Lexical Analyzer in C

A console-based Lexical Analyzer application developed using C programming language to analyze source code and identify various tokens such as keywords, identifiers, operators, constants, and special symbols.

## Features

* Identify keywords
* Detect identifiers
* Recognize operators
* Recognize special symbols
* Detect constants and literals
* Classify tokens
* Basic lexical error detection

## Technologies Used

* C Language
* File Handling
* String Manipulation
* Standard C Libraries
* Modular Programming
